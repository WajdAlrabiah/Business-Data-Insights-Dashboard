# Business Data Insights Dashboard

This project provides a **mini business data insight dashboard** that enables users to analyze and visualize business-related data.  
It combines **data cleaning, exploratory analysis**, and an **interactive Streamlit web app** for visual insights.

---

## Features
- Upload and preview your own CSV file.
- Automatic data cleaning (handling missing values, duplicates, and outliers).
- Compute mean, median, and mode.
- Visualize distributions using histograms.
- View correlation heatmaps between numerical features.
- Interactive Streamlit dashboard for user-friendly exploration.

---

## Technologies Used
- **Python 3**
- **Pandas** – Data manipulation and cleaning  
- **NumPy** – Numerical operations  
- **Matplotlib** & **Seaborn** – Data visualization  
- **Streamlit** – Interactive web interface
